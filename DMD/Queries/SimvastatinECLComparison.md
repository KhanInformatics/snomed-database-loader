## Recommendations

### For Clinical Systems:
1. **Use ECL for semantic searches** - captures full ontology including abstract concepts
2. **Use dm+d for prescribing** - contains only valid, prescribable UK products
3. **Map VTMs to VMPs** - when EMIS uses VTM codes, expand to VMPs for prescribing

### Search Strategy Comparison:

| Method | Query | Result Count | Use Case |
|--------|-------|--------------|----------|
| **ECL (broad)** | `<< 96304005` | 263 concepts | Semantic search, includes abstracts & international |
| **dm+d VMP name** | `WHERE nm LIKE '%Simvastatin%'` | 19 VMPs, 202 AMPs | UK prescribing, valid products only |
| **dm+d VTM expansion** | 4 VTM IDs → VMPs | 19 VMPs, 202 AMPs | Structured hierarchy navigation |

### EMIS SCT_CONST Implications:
- If EMIS code 390721000033117 maps to VTM 777537002, it would expand to 13 VMPs (single-ingredient only)
- Combination products (ezetimibe, fenofibrate, cholesterol) have separate VTMs
